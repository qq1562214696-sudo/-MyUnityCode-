# 项目脚本工具集

## 📋 概述
这是一套为3D美术和Unity开发优化的多平台效率工具集合，包含Unity、3ds Max、Blender、Photoshop和工作流自动化脚本。

## 🚀 各平台工具速览

### Unity编辑器工具 (C#)
**位置**: `Art/___创宇极___` 菜单
- **一键规范文件**: 批量修改FBX动画类型为Legacy，自动调整特定shader参数
- **批量导出**: 根据Max文件名自动导出匹配的预制体和文件夹到桌面
- **智能复制**: 复制选中对象并自动打开FBX导出窗口
- **Max文件名管理**: 列表化管理Max文件，支持粘贴、规范、导出

### 3ds Max工具 (MAXScript)
**位置**: 自定义QF工具精简版面板
- **规范模型**: 一键标准化材质、UV、轴心点
- **面数检查**: 自动检测并显示模型面数是否超标
- **多边面选择**: 快速选中五边及以上面
- **智能导出**: 自动识别角色部位并导出到Unity目录结构

### 文件夹整理工具 (PowerShell)
**运行方式**: 右键点击总文件夹运行
- **批量创建**: 基于模板快速生成道具文件夹结构
- **自动命名**: 重命名.max、贴图文件到统一格式
- **智能清理**: 可选择保留或删除源模板
- **剪贴板集成**: 自动复制创建的命名列表

### Blender批量转换器 (Python)
**位置**: 3D视图侧边栏 > ZW_Blender
- **格式转换**: 支持OBJ、FBX、GLTF等12+格式转FBX
- **批量处理**: 遍历子文件夹自动转换所有模型
- **日志系统**: 实时显示转换结果，支持临时日志文件
- **场景隔离**: 自动创建临时场景避免污染当前工程

### Photoshop侧边栏 (JavaScript)
**位置**: Photoshop脚本菜单
- **快捷面板**: 常驻侧边栏提供快速功能访问
- **图层管理**: 快速清空图层等常用操作
- **扩展框架**: 预留接口方便添加自定义功能

## 🎯 快速使用指南

1. **Unity工具**: 安装后从Art菜单打开，粘贴Max文件名列表 → 点击"一键规范"或"一键导出"
2. **3ds Max工具**: 运行脚本打开面板，使用"规范模型"标准化场景 → "导出"到Unity
3. **文件夹工具**: 右键点击包含Assets和截图的总文件夹 → 输入命名格式 → 自动创建
4. **Blender工具**: 安装插件后，在侧边栏选择文件夹 → 开始批量转换
5. **Photoshop工具**: 安装脚本后，从文件菜单运行显示侧边栏

## ⚙️ 安装说明

- **Unity**: 将C#脚本放入`Editor`文件夹
- **3ds Max**: 运行`QF_Tools_Lite`宏脚本
- **PowerShell**: 右键管理员运行`.ps1`文件
- **Blender**: 编辑 → 偏好设置 → 插件 → 安装插件
- **Photoshop**: 将`.jsx`文件放入Scripts文件夹

💡 **提示**: 所有工具设计为开箱即用，无需复杂配置。遇到问题请检查脚本控制台输出。
